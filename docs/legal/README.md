# Documentation juridique publique

Ce dossier contient uniquement les contrats techniques, rapports de validation
et checklists pouvant accompagner publiquement le code Clikarage.

Les documents de travail internes, hypotheses de conservation, procedures
operationnelles et sources mixtes ne font pas partie de l'arbre public. Ils ont
ete archives dans un espace prive distinct avant leur retrait. Leur ancienne
presence dans l'historique de la branche publique reste documentee ; aucune
affirmation n'est faite sur une absence d'acces anterieur.

Le corpus affiche par l'application provient exclusivement des modules
versionnes sous `src/features/legal/`. Les documents restent au statut
`staged` ou `draft`, sans date d'effet, tant qu'une validation juridique humaine
et une activation explicite n'ont pas eu lieu.

Ne jamais ajouter ici :

- un dossier `docs/legal/internal/` ;
- un document marque confidentiel, privilegie, brouillon interne ou non
  publiable ;
- une donnee personnelle non destinee aux mentions publiques ;
- un secret, token, cle ou detail de securite exploitable.
